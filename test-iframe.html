About Iframe Detection

The Moodle Tile Generator automatically detects when it's running inside an iframe and adjusts its appearance:

  • Hides the header: Removes the "Moodle Tile Image Creator" title to save vertical space
  • Reduces padding: Decreases container padding from 1rem/2rem to 0.5rem/1rem for more usable space
  • Detection method: Uses window.self !== window.top to detect iframe context

How to compare: Open the "Standalone View" link in a new tab to see the full header and normal padding, then compare with the "Embedded View" iframe on the right which has the header hidden and reduced padding.
